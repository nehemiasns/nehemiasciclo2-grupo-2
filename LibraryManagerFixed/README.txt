
LibraryManagerFX - Simplified JavaFX Library Management System

How to run:
- Requires Java 17+ and Maven.
- From the project root run: mvn javafx:run

This project stores data in JSON files at: ~/.libraryfx/
Default admin user: username=admin password=admin123

Features implemented:
- Login (basic authentication)
- Dashboard with counters
- Books management (add/edit/delete)
- Loans (register loan and return)
- Reports: export top users to CSV
- Backup of users.json

This is a simplified but complete runnable JavaFX application ready for further improvements.
